# UPLIFT Product Specifications

UPLIFT is a local-first AI Agent Operating System designed for orchestrating multiple AI agents on a user's devices. It offers privacy, control, and a unified interface for various workflows.

## Core Features

- **Local Desktop OS**: Run agents securely on your own hardware
- **Unified Chat Interface**: One place to command all your agents
- **Permissioned Data Sharing**: Controlled access between agents
- **Web2 + On-chain Integration**: Works across traditional and blockchain systems

## Target Audience

- **Traders**: Managing on-chain and exchange workflows
- **Builders**: Coordinating development, research, and project management
- **Operators**: Handling communications and team coordination

## Multi-OS Support

The roadmap includes support for:
- Mac
- Windows 
- iOS
- Android

## Vision

Provide a persistent "agent command center" across devices, evolving into the default OS layer for the agent era.
